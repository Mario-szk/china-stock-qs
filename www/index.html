PODA Chinese Stock Data Access Service Alpha 1

1. Stock transactions

URL: http://scisaga.net:50000/stock/transactions?code=<CODE>&sd=<SD>ed=<ED>&type=pandas_df_json

  • <CODE> is stock code, e.g., sh600036
  • <SD> is query start time, e.g., 2017-01-31
  • <ED> is query end time, e.g., 2017-02-03
  • Returned json is pandas friendly, you can load data by using pd.read_json(<URL>, orient='split')
    • URL:
	http://scisaga.net:50000/stock/transactions?code=sh600036&sd=2017-02-01&ed=2017-02-07&type=pandas_df_json
Return:
	{ "code": 1, "msg": "SUCCESS", "data": { "columns": [ "code", "price", "change_value", "volume", "turnover", "type" ],
	"index": [ "2017-02-03 09:30:00", "2017-02-03 09:30:03", ...],
	"data": [ [ "sh600036", 19.07, 19.07, 100, 1907.0, "买盘" ], ...]}

2. Stock time quotes

URL: http://scisaga.net:50000/stock/time_quotes?code=<CODE>&sd=<SD>ed=<ED>&type=pandas_df_json

  • <CODE> is stock code, e.g., sh600036
  • <SD> is query start time, e.g., 2017-01-31
  • <ED> is query end time, e.g., 2017-02-03
  • Returned json is pandas friendly, you can load data by using pd.read_json(<URL>, orient='split')
    • URL:
	http://127.0.0.1:50000/stock/time_quotes?code=sh600036&sd=2017-02-01&ed=2017-02-07&type=pandas_df_json
Return:
	{ "code": 1, "msg": "SUCCESS", "data": { "columns": [ "code", "price", "change_value", "volume" ],
	"index": [ "2017-02-03 09:30:00", "2017-02-03 09:31:00", "2017-02-03 09:32:00", ...],
	"data": [ [ "sh600036", 19.04, 0.04, 149400 ], ...]}
